# 💰 Personal Expense Tracker

A full-stack web application built with **Flask** that helps users efficiently manage their daily expenses, monitor spending patterns, and gain financial insights through an intuitive dashboard.

## 🚀 Features

- ➕ Add, edit, and delete expense records
- 📊 Real-time expense summaries and analytics
- 📅 Monthly spending reports
- 🗂️ Categorize expenses for better tracking
- 🔍 Search and filter expense history
- 💾 Secure data storage using SQLite/MySQL
- 📱 Responsive and user-friendly interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/expense-tracker.git
```

### Navigate to the project

```bash
cd expense-tracker
```

### Create a virtual environment (Recommended)

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

The application will be available at:

```
http://127.0.0.1:5000/
```
